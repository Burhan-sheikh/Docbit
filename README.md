# Docbit

Docbit is a modular SaaS-style document platform built with React + TypeScript + Vite.

## Features in this scaffold

- Responsive tool grid dashboard (2/3/4-6 columns).
- Unified workspace with top bar, preview area, bottom-sheet controls, and floating action button.
- Centralized Zustand store for files/settings/processing state.
- Worker-based processing simulation with progress and task queue status.
- Tailwind dark premium UI foundation with motion interactions.

## Run

```bash
npm install
npm run dev
```
